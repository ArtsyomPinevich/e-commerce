## How to use 
1. Clone repository 
```
git clone https://github.com/ArtsyomPinevich/e-commerce.git
```
2. open project folder 
```
cd e-commerce
```
3. install dependencies: 
```
npm i
```
4. run dev server 
```
npm run dev
```
